O que vou aprender?

Nesta unidade curricular vamos aprender HTML, CSS e JavaScript que trabalham juntos para formar uma página web.

  • HTML: Cria a estrutura da página.
  • CSS: Cria a formatação da página.
  • JavaScript: Cria a interatividade da página.

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas web. Ele é responsável por estruturar o conteúdo na internet, como textos, imagens, links e outros elementos. Basicamente, o HTML define a estrutura básica de um site e informa ao navegador como exibir os elementos na tela.

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar e definir a aparência das páginas web. Enquanto o HTML estrutura o conteúdo da página, o CSS controla como esses elementos são exibidos, como cores, fontes, tamanhos, espaçamento, e disposição na tela.
